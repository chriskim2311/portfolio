Finance Tracker

Category : Web-Design / Apps

Apps

Live | Github

Role: Solo Project, Full Stack Developer

Responsibilities: On the front-end, I created a fully responsive tracker of receipts with CRUD operations. On the back-end, I created a MySQL database with PHP CRUD back-end calls.

Tools and Tech: JavaScript, PHP, MySQL, jQuery, HTML, CSS, BootStrap

Finance Tracker is a receipt tracking app that allows for manual data input that totals and renders the data from the receipt. This app was my first full stack app where I built the entire database and rendered all data on the DOM using jQuery, BootStrap and JavaScript.
